Course Objectives

In this course, the followings will be covered to adequately enable web programming and web application development:

  • Understand the protocols, language and systems used on the Web (HTML, CSS, HTTP, URLs, XML)
  • Understand the functions of clients and servers on the Web & learn how to implement client-side scripts (JavaScript) and server-side scripts (PHP)
  • Obtain ability to design and implement an interactive web site where usability, accessibility and internationalization issues are considered
  • Learn how to use/manage database associated with web applications (mainly storage and retrieval)

Assesment

  • Attendance (10%)
  • Midterm Exam (30%)
  • Final Exam (30%)
  • Team Project (30%)
